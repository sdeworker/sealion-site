产品中心

挤出产线的测控系统

从进料称重到质量入库，从单机控制到挤出云——覆盖塑料挤出全流程的八套核心测控系统。

01

米重控制系统

Gravimetric Control System

安装于挤出机进料口，采用 PID 闭环控制，实时精准控制每米克重，稳定管材质量、节省 2%–5% 原料。

02

米重色母控制系统

Gravimetric Masterbatch Control System

在控制米重的同时自动配比色母添加，色母机随主机动态变化，微积分闭环，配比精度 ±1%。

查看详情 →
03

米重色母系统称重版

Gravimetric Masterbatch Weighting Control System

色母以失重法精确称量计量，多组分独立控制，配方可存档追溯，适配单/双/三组份添加。

查看详情 →
04

超声波在线测厚系统（大管）

Ultrasonic Thickness Online (Big Pipe)

超声波非接触在线测量管材壁厚、偏心、椭圆与外径，测量分辨率达 0.01mm，适配 Φ63–1200 大口径管材。

查看详情 →
05

超声波在线测厚系统（小管）

Ultrasonic Thickness Online (Small Tube)

面向小管径的在线测厚，独创自动变径结构，探头在线自动对中跟随，测壁厚 0.18–10mm、直径 2–50mm。

查看详情 →
06

管材质量安全入库系统

Pipe Inbound Quality & Safety System

挤出机末端在线全检：激光测厚、编码器测径、不圆度、重量、长度，合格翻管入库，数据接入 MES。

查看详情 →
07

管材在线智能检测系统

Online Intelligent Pipe Inspection System

产线末端自动检测平台，激光测厚、测径、不圆度、合格翻管一体，检测数据实时上传，远程监控管材质量。
