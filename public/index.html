产品中心

挤出产线的测控系统

从称重到上云——覆盖挤出全流程的七道工序：称重 → 测量 → 喂料 → 配料 → 一出二 → 质检 → 数字化。

01

米重色母称重控制系统

Gravimetric Control System

安装于挤出机进料口，基于称重与 PID 闭环，实时控制每米克重与色母配比，稳定产品重量、降低原料与色母超耗。

02

超声波在线测厚系统

On-Line Ultrasonic Thickness

非接触超声波在线测量并记录管材壁厚、内径与外径，超差即时报警，让壁厚在运行中看得见。

03

失重式喂料机系统

Loss-in-Weight Feeder

通过持续称量料仓失重计量物料，向挤出机提供稳定精确的质量流，自动补料不中断。

04

多组分混料系统

Multi-Component Mixture

按配比在线自动配混多种物料，自适应算法保证配比精度、混料均匀，适配薄膜、电缆、管材与改性料。

05

一出二控制系统

Dual-Line Extrusion Control

面向一模多出工艺，协调控制多出线的重量与速度，兼顾产能与一致性。

06

管材表面缺陷检测系统

Surface Defect Inspection

机器视觉实时检测管材表面缺陷，即时标记，质量数据可接入产线系统。

07

管材挤出云远程监控系统

Cloud Remote Monitoring

汇聚产线重量、测厚、质检数据上云，远程实时监控产能与质量，支持多线管理。
